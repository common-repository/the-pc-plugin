== Description ==

Turns any blog into a haven of safe PC content. 


== Installation ==


1. Upload `pcplugin.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Feel free to add your own vocabulary to the plugin by editing the php file.
